Description

Event flow

Main Flow

  1. User introduces his/her new username.
  2. User introduces twice the new pasword.
  3. System checks if the credentials are correct and creates account to the user.
  4. System redirecs the user to the Log in window of the app.

Alternative flow

A1. If the username introduced by the user is taken, system displayes an error message to the User.

B2. If the passwords introduced by the user are not equal, sytem displays an error message to the user.
