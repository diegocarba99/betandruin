Extract earnings

UMLUseCase
Bet&Ruin :: Use Case View :: Extract earnings

Description

Event flow

Main Flow

  1. User enters the amount of money he/she wants to withdraw from his/her Virtual Wallet's funds.
  2. User confirms the action via a button in the screen.
  3. System displays a confirmation message.
  4. User can accept the transaction and ge the corrresponding Ruin Points or cancel the whole proccess.

Alternative flow

A1. If the user types any non-digit and non-positive amount the system won't introduce those characters.

B2. If the user doesn't confirm the action, system return to step 1.

Prequisites

User must have accessed the Virtual Wallet management view.
