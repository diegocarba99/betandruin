Manage data

UMLUseCase
Bet&Ruin :: Use Case View :: Manage data

Description

Event flow

Main Flow

  1. Admin selects the element (s)he wants to manage (Users, Bets, Funds movements, Events).
  2. System shows in the window all the elements ordered.
  3. Admin clicks the element he wants to manage.
  4. System redirects the Admin to a window related to the object (s)he has clicked.

Alternative flow

A1. Admin can search for specific object by id, name, ...

A2. Admin can order objects as desired by id, name, date, ...
