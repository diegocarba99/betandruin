Event flow

Main Flow

  1. Admin clicks "Delete" button.
  2. System asks for confirmation.
  3. Admin confirms by clicking "Yes" button.
  4. System deletes Event from the data base.

Alternative flow

A1. Admin cancells confirmation by clicking "No" button.

Prequisites

Admin has double-clicked an Event in the list.
