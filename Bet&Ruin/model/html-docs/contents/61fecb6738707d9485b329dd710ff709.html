Browse bets

UMLUseCase
Bet&Ruin :: Use Case View :: Browse bets

Description

Event flow

Main Flow

  1. User selects a Sport and a Category of this Sport.
  2. System displays the events of this sport.
  3. User selects either one of the posibilities of the main bet
  4. System displays the summary asssociated to selected bet.

Alternative flow

A2. If there are no events to be displayed, then events cannot be shown. Sytem displays a message to the User.

B4. If there are no questions associated to the event, then questions cannot be shown. A message is shown to the User.

Prequisites

User must had been logged in before browsing the bets.
